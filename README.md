## Hotel Paraíso v2.1

### DESCRIPCIÓN

▪ Hotel Paraíso es un proyecto para el grado. Una web simulando un hotel vacacional lacalizado en la isla de Lanzarote(España). 
*No es una web real*. Realizado sin JS, compuesto unicamente por HTML y CSS.
